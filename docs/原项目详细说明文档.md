# Barra风格因子择时策略 - 详细说明文档

## 1. 项目概述

本项目实现了一个基于Barra CNE6模型的风格因子择时策略：

1. **因子择时**：通过因子累计收益率的通道突破信号判断因子走势
2. **最优向量**：根据择时信号生成最优因子暴露向量
3. **股票选择**：在全市场选择因子暴露与最优向量最相似的100只股票
4. **周度换仓**：按周切换时点换仓，使用前一交易日数据生成信号（避免偷看未来数据）

## 2. Barra CNE6模型

### 25个风格因子

| 类别 | 因子 | 说明 |
|------|------|------|
| 规模 | SIZE, MIDCAP | 大盘股vs小盘股，中市值效应 |
| 波动率 | BETA, RESVOL, HISTVOL | 系统风险，特质波动，历史波动 |
| 动量 | MOMENTUM, RESMOM | 价格趋势，特质动量 |
| 价值 | BTOP, EYIELD, CFP, SP, LP | 估值因子 |
| 成长 | EGRO, SGRO | 盈利增长，收入增长 |
| 杠杆 | MLEV, BLEV | 市场杠杆，账面杠杆 |
| 流动性 | LIQUIDITY, STOM, STOQ | 换手率因子 |
| 盈利质量 | ROE, ROA | 盈利能力 |
| 投资 | CAPX, AGRO | 资本支出，资产增长 |
| 其他 | TOPSI, SEASON | 极端换手，季节性 |

### 因子计算示例

```python
# SIZE因子
SIZE = log(total_market_cap)

# BETA因子
model = OLS(stock_returns, market_returns)
BETA = model.params[0]

# MOMENTUM因子（跳过最近1个月）
MOMENTUM = (price.shift(21) / price.shift(252)) - 1

# ROE因子
ROE = net_income / book_equity
```

## 3. 择时信号生成

### 通道分档算法

```python
def calc_pst(close, long_prd=20, short_prd=5, channel_bins=3):
    """
    计算择时信号
    
    参数:
        close: 收盘价序列（因子累计收益率）
        long_prd: 长周期窗口（默认20）
        short_prd: 短周期偏移（默认5）
        extreme_value: 极端值（默认3）
    """
    # 计算通道上下轨
    bf_max = close.shift(short_prd).rolling(long_prd).max()  # 上轨
    bf_min = close.shift(short_prd).rolling(long_prd).min()  # 下轨
    
    # 计算通道位置（0=下轨, 1=上轨）
    position = (close - bf_min) / (bf_max - bf_min)

    # 生成信号：通道外为±1，通道内按channel_bins离散分档映射到(-1,1)
    signal = pd.Series(np.nan, index=close.index)
    signal[position > 1] = 1
    signal[position < 0] = -1
    # 其余在[0,1]区间内做等分分档
    
    return signal
```

**为什么使用 `shift(short_prd)`？**

避免使用未来数据。假设今天是第100天：
- `bf_max = close[75:95].max()` - 使用第75-95天数据，不包含第96-100天
- 这样今天的信号只依赖历史数据

### 最优向量生成

| 信号值 | 含义 | 操作 |
|--------|------|------|
| +1 | 通道上方 | 做多该因子（选择高暴露股票） |
| (-1, 1)离散档位 | 通道内 | 按档位强弱配置 |
| -1 | 通道下方 | 做空该因子（选择低暴露股票） |

最优向量通过 `optimal_vector = signal * extreme_value` 生成。

## 4. 股票选择算法

### 余弦相似度

```python
def cosine_similarity(v1, v2):
    """
    计算两个向量的余弦相似度
    
    公式: cos(θ) = (v1 · v2) / (||v1|| * ||v2||)
    """
    dot_product = np.dot(v1, v2)
    norm_v1 = np.linalg.norm(v1)
    norm_v2 = np.linalg.norm(v2)
    
    return dot_product / (norm_v1 * norm_v2) if norm_v1 and norm_v2 else 0
```

**示例：**

```
最优向量: [3, -3, 0, 3, ...]  # SIZE=3, MOMENTUM=-3, BETA=0, ROE=3

股票A因子暴露: [1.5, -1.2, 0.5, 2.0, ...]
股票B因子暴露: [-0.8, 0.9, 0.3, -1.5, ...]

similarity_A = 0.85  → 选择股票A
similarity_B = -0.72
```

### 选股流程

1. 获取信号日期的因子暴露（避免偷看未来数据）
2. 过滤股票池（剔除ST、停牌、新股等）
3. 计算每只股票与最优向量的相似度
4. 按相似度降序排序，选择Top 100

### 预计算周度收益率

**问题：** v1/v2每次换仓都要计算周度收益率，复杂度O(112亿次)

**解决方案：**

```python
def precompute_weekly_returns(self, price_df, weekly_dates):
    """预先计算所有周度收益率"""
    weekly_returns = {}
    
    for i in range(len(weekly_dates) - 1):
        start_date = weekly_dates[i]
        end_date = weekly_dates[i + 1]
        
        # 向量化计算周度收益率
        week_prices = price_df[
            (price_df['date'] >= start_date) & 
            (price_df['date'] < end_date)
        ]
        
        weekly_return = week_prices.groupby('stock_code')['close'].last() / \
                        week_prices.groupby('stock_code')['close'].first() - 1
        
        weekly_returns[end_date] = weekly_return
    
    return pd.DataFrame(weekly_returns)
```

**性能提升：** 147倍（从30分钟降至1分钟以内）

## 6. 关键问题与解决方案

### 问题1：股票池过小（97只）

**原因：** 使用 `dropna()` 删除了任何因子缺失的股票

```python
# ❌ 错误做法
factor_exposure_df = factor_exposure_df.dropna(subset=self.style_factors)
```

**解决方案：** 填充缺失值

```python
# ✅ 正确做法
for factor in self.style_factors:
    factor_exposure_df[factor] = factor_exposure_df.groupby('date')[factor].transform(
        lambda x: x.fillna(x.mean())
    )
    factor_exposure_df[factor] = factor_exposure_df[factor].fillna(0)
```

**效果：** 股票池从97只增加到5388只

### 问题2：偷看未来数据

**原因：** 换仓时使用了当日的数据

```python
# ❌ 错误做法
signal_date = trade_date  # 使用当天的数据
```

**解决方案：** 使用前一天的数据

```python
# ✅ 正确做法
signal_date = get_previous_trading_day(trade_date)  # 前一交易日
```

### 问题3：日期类型不匹配

**原因：** DataFrame的date列是字符串，比较值是Timestamp

**解决方案：**

```python
factor_exposure_df['date'] = pd.to_datetime(factor_exposure_df['date'])
signal_date = pd.to_datetime(signal_date)
```

### 问题4：周度收益率矩阵为空

**原因：** 使用精确匹配，如果日期不存在则返回空

**解决方案：** 使用最近日期匹配

